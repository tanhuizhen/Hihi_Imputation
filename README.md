Pipelines and analysis codes for the following paper: Tan, H. Z., Stuart, K. C., Vi, T., Whibley, A., Bailey, S., Brekke, P., Santure, A. W. (2025). High imputation accuracy can be achieved using a small reference panel in a natural population with low genetic diversity. Molecular Ecology Resources https://doi.org/10.1111/1755-0998.70024

Folder names correspond to the sections in 'Materials and methods'.

Linkage map results showing the map positions of each marker is accessible at: https://github.com/tanhuizhen/Hihi_Linkage-mapping/tree/main/Results_hihi_linkage_map

Unless otherwise stated, codes were ran via Jupyter on New Zealand eScience Infrastructure (NeSI).
